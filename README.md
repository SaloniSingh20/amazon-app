ShopSphere is a full-stack e-commerce web application inspired by Amazon,
redesigned and extended with a focus on clean architecture, scalability,
and real-world shopping features.

This project demonstrates end-to-end development using the MERN stack,
including user authentication, product management, and order handling.

## Features

- User authentication with JWT (Login / Register)
- Product listing with detailed product pages
- Add to cart and cart management
- Order placement and order history
- Responsive UI for desktop and mobile devices
- Secure REST APIs for backend operations

## Improvements Made

- Refactored backend into modular MVC architecture
- Improved UI/UX for better user experience
- Added input validation and centralized error handling
- Optimized API calls and state management
- Improved folder structure for scalability and maintainability

## Tech Stack

**Frontend**
- React.js
- JavaScript (ES6+)
- HTML5, CSS3

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB (Mongoose)

**Other Tools**
- JWT for authentication
- Git & GitHub for version control
